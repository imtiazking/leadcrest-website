# Leadcrest - Dubai Real Estate Lead Generation

A professional lead generation website for Leadcrest Group, delivering pre-verified qualified buyers in Dubai.

## Project Structure

```
leadcrest-website/
├── index.html              # Main landing page
├── leadcrest-logo.png      # Main logo (new design)
├── logo.png                # Original logo
├── logo.jpeg               # Backup logo
├── beach.mp4               # Hero video background
├── how-it-works/           # Secondary pages
│   └── index.html
├── vercel.json             # Vercel deployment config
├── .gitignore              # Git ignore rules
└── README.md               # This file
```

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Hero section with video background
- ✅ Stats ticker with key metrics
- ✅ How It Works section with step breakdown
- ✅ Lead Quality verification details
- ✅ Pricing tables (Pay-Per-Lead & Hybrid Retainer)
- ✅ FAQ accordion
- ✅ Contact form (CTA)
- ✅ Professional footer

## Deployment

### Vercel (Recommended)
1. Push to GitHub
2. Connect repo to Vercel at https://vercel.com
3. Auto-deploys on every push

### Netlify
1. Upload files via dashboard
2. Or connect GitHub repository
3. Auto-deploys on push

### Local Development
Simply open `index.html` in a browser to view locally.
